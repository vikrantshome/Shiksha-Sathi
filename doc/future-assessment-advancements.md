# Future Assessment Advancements
**Note for Future Development**

Based on initial research and requirements gathering, the following features have been identified as high-value additions to the assessment system but are deferred to future iterations to focus on core worksheet stability.

## 1. Inline Feedback Threads
Allow teachers to start a feedback thread directly within a worksheet cell.
- Support for rich-text comments.
- Notifications for students when feedback is added.
- Student-teacher reply chains.

## 2. Advanced Grading Rubrics
Integrate scoring rubrics directly into the spreadsheet headers.
- Visual display of rubric criteria on cell focus.
- One-click grading based on rubric levels (e.g., "Exceeds Expectations", "Meets Expectations").

## 3. Real-time Collaborative Grading
Allow multiple teachers/admins to work in the same worksheet simultaneously with visual presence indicators (like Google Sheets cursors).

## 4. Rich Media Submissions & Grading
Support for grading handwritten uploads, voice notes, and video submissions directly within the worksheet grid via hover-previews.
